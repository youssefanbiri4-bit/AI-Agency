# Final Testing Checklist

## Alex

- Ask Alex for a campaign plan.
- Ask Alex for an n8n plan.
- Ask Alex for code/debugging help.
- Open `/dashboard/alex?template=<template-id>`.
- Confirm selected template panel is clean.
- Confirm Smart Recommendations are useful and not crowded.
- Confirm chat messages scroll independently.
- Confirm input stays fixed/sticky at the bottom.
- Copy prompt/response and verify clean formatting.
- Test Arabic/Darija input and confirm RTL readability.

## Agent Library

- Search templates.
- Filter every category.
- Copy Prompt.
- Use with Alex.
- Create Task and confirm pending status.
- Send to Content Studio.
- Export n8n Workflow Plan.
- Confirm usage analytics failure does not break the page.

## Content Studio

- Open `/dashboard/content-studio?template=<template-id>`.
- Confirm fields are prefilled.
- Confirm no AI generation starts automatically.
- Confirm no save/publish/schedule starts automatically.
- Copy generated output and verify formatting.

## Workflow Builder

- Build workflow from preset.
- Add/remove templates.
- Reorder steps.
- Confirm visual diagram appears first.
- Review workflow.
- Preview dry run.
- Export Markdown.
- Create pending tasks only after confirmation.

## Playbooks

- Save a workflow as a playbook.
- Open the saved playbook.
- Duplicate it.
- Favorite/unfavorite it.
- Search/filter/sort playbooks.
- Export Markdown.
- Create pending tasks with confirmation.
- Delete and confirm it only removes the playbook.

## Safety

- Confirm no n8n auto-run.
- Confirm no provider publish.
- Confirm no scheduling.
- Confirm no live ads.
- Confirm no spending.
- Confirm no secrets in client output or exported text.
